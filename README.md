# kintone-js-sdk
REST API client of kintone for node.js and javascript 

# Installation
npm i @kintone/kintone-js-sdk

# Usage
* UMD: include <i>kintone-js-sdk.min.js</i> from <i>dist</i> folder <br/>
* CommonJS: <br/>
```
const kintone = require('kintone-js-sdk')
```
